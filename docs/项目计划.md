# 📈 项目演进计划

本项目已成功实现视频采集、硬件编码与网络推流的完整框架。

---

## 🛠️ 当前里程碑 (已完成)

- [x] **高性能采集链路**: 深度整合 ISP -> VI -> VENC。
- [x] **RTSP 推流集成**: 支持多路 RTSP 实时预览。
- [x] **双码流并行编码**: 支持主流 (1080P) 与子流并行。
- [x] **动态参数管理**: 引入 `rk_param` 与 INI 配置文件。
- [x] **命令行解析**: 支持运行时配置文件与日志级别切换。
- [x] **资源自动化管理**: 完善的模块初始化与反初始化。
- [x] **线程架构重构**: 实现 采集/编码/推流 的多线程流水线分离。
- [x] **RTMP 云端推流**: 集成官方 rkmuxer，支持推流到云平台。
- [x] **RGA 硬件加速**: 集成 2D 图形加速库，支持高效缩放与裁剪。
- [x] **OSD 视觉叠加**: 集成 FreeType，实现实时时间戳与文字叠加。
- [x] **性能监控**: 实时监控 CPU、内存、温度及视频编码性能。

---

## 🏗️ 近期路线图 (计划中)

### 1. 多媒体进阶 🎶
- [ ] **音频系统**: 增加音频采集 (ALSA) 与编码 (AAC/G.711)。
- [ ] **音视频同步**: 实现 RTSP 流的音视频同步输出。
- [ ] **抓拍优化**: 支持高分辨率原图抓拍。

### 2. 智能化与交互 🧠
- [ ] **OSD 增强**: 实现实时时间戳、自定义 Logo 与动态文字叠加。
- [ ] **AI 插件化**: 预留 NPU 接口，支持物体识别结果的边缘绘制。
- [ ] **Web 控制台**: 增加简单的 HTTP 服务，支持网页端修改 INI 配置。

### 3. 系统稳定性 🛡️
- [ ] **看门狗集成**: 防止程序长时间运行异常。
- [ ] **存储管理**: 实现循环录像功能，支持 SD 卡自动分段存储。
