# ⚡ 快速开始指南

本指南将指导您如何在几分钟内完成项目的编译、部署与运行。

---

## 🛠️ 1. 编译项目

在开发主机上执行以下命令进行交叉编译：

```bash
./build.sh
```

**编译结果：**
- 可执行文件路径：`build/rv_demo`

---

## 📦 2. 部署到板端

使用 `adb` 或 `scp` 将编译生成的程序及配置文件拷贝到 RV1126 开发板上：

```bash
# 拷贝可执行文件
adb push build/rv_demo /userdata/
# 拷贝配置文件 (可选，程序有默认路径)
adb push rkipc.ini /userdata/
```

---

## 🏃 3. 运行程序

进入板端终端，可以使用多种参数启动程序：

```bash
# 默认启动 (加载 /userdata/rkipc.ini)
./rv_demo

# 指定配置文件与日志级别启动
./rv_demo -c /userdata/my_config.ini -l 3
```

---

## 🔍 4. 验证输出

确认程序是否成功启动并输出了 RTSP 地址（查看控制台日志）：

```text
[INFO] rkipc init finished.
[INFO] RTSP server started at rtsp://<IP>/live/0
```

---

## 📺 5. 实时预览

1. **确保您的 PC 与开发板在同一网段。**
2. **使用 ffplay 查看视频流：**
   ```bash
   ffplay rtsp://<开发板IP>/live/0
   ```

---

## ⚙️ 6. 修改配置

本项目支持通过 **INI 配置文件** 动态调整参数，无需重新编译。

- **配置文件**: `/userdata/rkipc.ini`
- **主要参数**:
  - `[video.0]`: 分辨率、码率、帧率、编码格式 (H.264/H.265)。
  - `[isp.0.adjustment]`: 曝光、增益、FPS 等。
  - `[rtsp]`: 推流路径设置。

修改后重启 `rv_demo` 即可生效。
