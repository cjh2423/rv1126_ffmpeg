# 快速开始

## 1. 编译

```bash
./build.sh
```

输出：`build/rv_demo`

## 2. 部署到板端
将 `build/rv_demo` 拷贝到板端可执行路径即可。

## 3. 运行

```bash
./run.sh
```

## 4. 验证输出
默认输出裸码流：

```bash
ls -lh /tmp/rv_demo.h264
```

## 5. 拉取与播放

```bash
adb pull /tmp/rv_demo.h264 ./
ffplay -f h264 -framerate 30 rv_demo.h264
```

## 6. 修改参数
修改 `main/config/config.h`：
- 分辨率/帧率/码率/GOP
- 编码格式（H.264/H.265）
- 输出文件路径
